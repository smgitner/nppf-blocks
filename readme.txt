Reusable Gutenberg blocks for NPPF website sections.

== Description ==

NPPF Blocks provides a collection of custom Gutenberg blocks designed for the National Press Photographers Foundation website. Each block is fully customizable and responsive.

**Included Blocks:**
- Hero Section
- Scholarship Grid
- What We Do
- Impact Stats
- Testimonial
- Call to Action
- Footer

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/nppf-blocks`
2. Activate the plugin through 'Plugins' menu in WordPress
3. Uncomment the activation line in nppf-blocks.php
4. Use the blocks in the Gutenberg editor under "NPPF Blocks" category
